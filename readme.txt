说明：本MOD为完全革新MOD，三国杀中的核心牌多已被换去，因此对于期望能够轻轻松松地马上开始玩的朋友们，额，ctrl + w 应该是个很方便的快捷键。

简单的说明：

本MOD基于神杀重新制作了一套104张游戏牌的卡牌，对原有系统进行了置换与添加；但是基本的游戏流程与游戏方式未改变，你可以将此MOD看做是一套全新的“环境”。

新概念与新系统：

一、灵力系统

在本MOD中，每名玩家除了体力值之外还具有灵力值。与体力一样，每个武将都有其特定的灵力上限，但游戏开始时每名玩家的灵力值都是0。回合结束之后，玩家将自然回复一点灵力。

灵力的作用有2：距离的突破和技能的发动。

本作的所有有距离限制的游戏牌（战斗牌、春雪异变和“重要的东西被偷走了！”）（其实就是杀、兵粮寸断和顺手牵羊），都可以通过消耗灵力来增加使用范围。

拿顺手牵羊来说，如果你到对方的距离为2，本来是无法对对方使用的；但是如果你有一点灵力，顺手牵羊就可以依然对对方使用，但是会消耗一点灵力。

本作中的武器距离略短，最长的神枪冈格尼尔也只有3，因此活用灵力突破距离是非常重要的。

灵力还可以用于技能发动。不过，鉴于目前制作完成的武将只有4个，这一点还无法完全体现。

二、战斗牌系统

本MOD取消了杀与闪的战斗系统，并以半原创的所谓“战斗牌系统”代替。

简单来说，就是牌堆里面一张杀或闪都没有，但是多了三种牌：弹幕、灵击与符咒。这三种牌互相克制。

这三种战斗牌都可以用来进攻，且每回合总共只能打出一张，就像杀一样。如果对方能够打出克制你的战斗牌，那么就不会受到伤害，就像打出了闪一样；如果不能，就会受到损失。

与原作不一样的是，你打出战斗牌时背面朝上，对方响应的牌也如此。当双方都打出了牌之后，就展示双方的战斗牌，并计算战斗结果。所以说这个系统，其实类似石头剪子布。

三、冻结与束缚

本作中大量使用了两个原创系统：冻结与束缚

冻结是鸡肋的推广；鸡肋只指定一种类型的牌直到回合结束，而冻结对某一张牌起作用，并且会持续到这张牌的所有者的回合结束。

冻结除手牌外的其他牌时，通常会描述为“返回手牌并冻结”。

另外还有一种“暂时冻结”，和冻结效果一样，但是只持续到当前玩家回合结束；不过，如果你同时有任何一张其他的牌被“冻结”到了你的回合结束，那么被“暂时冻结”的牌也会跟着被冻结到那个时候。

束缚是一种类似于“状态附加”的技能。对某玩家施放了束缚之后，他会在一定时间内获得一个具有负面效果的锁定技，而他的头像会有铁锁提示。将鼠标悬停在其头像上方，你可以看到这个负面技能的具体效果。

四、丰富的马

本作中每一匹马，除+1与-1外，都具有一些特殊效果。
它们大部分都允许你将已经装备的马弃置，然后发动某个技能，就像欢乐包的“猴子”一样。

五、开发进度

所有的代码和绘图都由自己设计完成，所以非常慢。

代码方面，目前做完了所有104张游戏牌，和博丽灵梦、藤原妹红、射命丸文、琪露诺四名武将的代码。由于事件体系已经大致完成，之后即使无所避免地要改也不会很麻烦。

由于武将太少，目前我还没有屏蔽原版武将和游戏牌。

/-------------------------------------------新扩展-----------------------------------------/

1、新战斗牌【毛玉】

毛玉克制所有其他种类的战斗牌，但是被所有非战斗牌克制。（被非战斗牌克制时，对方回复1点灵力//未做）
毛玉的进攻和防御效果都是回复1点灵力。

毛玉的数目和花色点数分布并没有确定，但是确定不会很多，大概5张左右。


2、天气

天气为一种延时判定牌，可以对任何一名玩家使用。使用时，场上存在的其他天气立即被弃置。
天气类似闪电，会在所有玩家面前传递；但是，天气的生效条件没有闪电苛刻，并且即使生效也不会被弃置。

3、结界

结界同样为延时判定牌，但结界在判定阶段并不进行判定。
其所有者通过在需要的时候弃掉结界来发动其效果。

本质为一次性的装备牌。

可能会有一些依附条件 - 如果你没有达到这些条件，结界立即被弃置。
